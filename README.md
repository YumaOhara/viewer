# First
```
# SDL2 (ウィンドウとイベント管理) の開発用ライブラリ
sudo apt-get install libsdl2-dev

# libtiff (TIFF画像読み込み) の開発用ライブラリ
sudo apt-get install libtiff-dev

# OpenGL の開発用ライブラリ
sudo apt-get install libgl1-mesa-dev
```

# CMake
```
mkdir build
cd build
cmake ..
cmake --build .
```

# How to use
`./build/viewer /pass/to/image/file`
than viewer starts
